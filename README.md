# 🌸 Ana's Blossom House

Ana's Blossom House is a responsive flower shop website built with HTML, CSS, JavaScript, Bootstrap, and Node.js. The project features a clean and elegant user interface that showcases a variety of flowers while providing a smooth and user-friendly browsing experience.

## 🌐 Live Demo

🔗 https://anas-blossom-house.netlify.app/

---

## ✨ Features

- 🌷 Responsive design
- 💐 Beautiful flower collection
- 🎨 Modern and minimal UI
- 📱 Mobile-friendly layout
- ⚡ Interactive elements using JavaScript
- 🖥️ Backend support using Node.js

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Bootstrap
- Node.js

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/anaghaajoshii-arch/Anagha.git
```

2. Open the project folder.

3. Install dependencies (if required)

```bash
npm install
```

4. Start the backend server

```bash
node backend/server.js
```

5. Open `index.html` in your browser.
